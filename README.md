## Project Overview
This is Part 1 of the Izzy Burger website project. It is a static website for a burger fast-food takeaway that showcases the menu, provides information about the restaurant, and allows customers to contact the business.

## Website Goals and Objectives
- Creates a professional online presence for Izzy burger
- Display the restaurant menu in a clear and accessible way
- Provide customer contact information
- Build a responsive and use-friendly layout

## Key features and functionality
- Home page with welcome message and branding
- Menu page listing burger options
- About page with company background
- Contact pagee with contact form
- Order Now page for customer orders
